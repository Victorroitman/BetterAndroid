# 基本配置

这里介绍了 BetterAndroid 可能会用到的项目功能的配置方法。

# View Binding

如果你引用了 ui-component 或 ui-extension 相关功能， 你需要在 buildFeatures 中启用 viewBinding。

android {
    buildFeatures {
        viewBinding = true
    }
}

# Jetpack Compose

如果你引用了 compose-extension 或 compose-multiplatform 相关功能，你需要在 buildFeatures 中启用 compose。

请参考 这里在新窗口中打开 设置正确的 kotlinCompilerExtensionVersion。

android {
    composeOptions {
        kotlinCompilerExtensionVersion = "/** Compose Compiler Version */"
    }
    buildFeatures {
        compose = true
    }
}
